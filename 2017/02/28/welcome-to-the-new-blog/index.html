Welcome to the New Blog

2017-02-28
extendvs, visual studio

It’s been a little while coming, but at last it’s finally here!

I plan to blog mainly on Visual Studio topics; any tips, tricks, gotchas, extensions, observations or cool techniques that myself or someone in the community might be using. A good source of knowledge for these blog posts will be the Microsoft/extendvs gitter room.

Sharing the visual studio love so to speak.

After looking around at a few different blogging platforms, I decided to use a static blog generator called Hexo. I’m using a Git repository, and hosting it on GitHub Pages for free. I’ve even created a custom Hexo theme.
